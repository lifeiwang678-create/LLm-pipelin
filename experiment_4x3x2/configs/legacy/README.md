# Legacy Configs

These configs are kept for reference only. They are not part of the maintained
4 x 3 x 2 prompt-compatible experiment path.

The current official LM usage registry supports:

- `direct`
- `few_shot`
- `multi_agent`

Configs in this folder may reference older experimental LM types such as
`sensorllm_checkpoint`, which are not registered in the current official
pipeline.
